查看本机ssh公钥，生成公钥

查看ssh公钥方法：

1.通过命令窗口

a.打开你的git bash 窗口
b.进入.ssh目录： cd ~/.ssh
c.找到id_rsa.pub文件： ls

d.查看公钥： cat id_rsa.pub 或者 vim id_rsa.pub
如图：
2、或者直接一步输入命令： cat ~/.ssh/id_rsa.pub
3、或者直接在C:\Users\user.ssh文件夹下查找
说明： 在linux系统和window系统下都可以使用命令行查看 （注意：window系统，需要在git bash下使用）
